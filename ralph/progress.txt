## Codebase Patterns
- `getComboColor(level)` returns combo-tier color: x1='#FF8800', x2='#FFD700', x3='#00FFDD', x4+='#FF44FF' — reuse for all combo-colored rendering
- Near miss tracking vars (`nearMissCount`, `bestCombo`, `nearMissBonusTotal`) are module-scope, reset in `resetGameState()`
- To apply combo multiplier to a score source: `* (comboMultiplier >= 2 ? comboMultiplier : 1)` — SURVIVOR_BONUS is intentionally excluded
- `triggerNearMiss(v)` increments combo BEFORE tracking bestCombo, so bestCombo reflects the new level
- `spawnFloatText(x, y, text, color, scale, duration)` — 6th param overrides FLOAT_TEXT_DURATION for hints/tutorials
- `triggerShake(duration, intensity)` has priority guard — weaker shakes don't override stronger ones in progress
- Particle shape: `{ x, y, vx, vy, life, maxLife, color, size }` + optional `isNitro` — add new types without new fields
- AudioManager SFX pattern: check `this.ctx`, create osc+gain, connect to `this.masterGain`, schedule, cleanup onended
- `gameOverState.onEnter()` captures module-scope vars as `this.finalXxx` for display
- Combo HUD expire effects use separate timers (`comboExpireShakeTimer`, `comboExpireFadeTimer`) triggered when `comboResetTimer` hits 0
- Milestone flash (`comboMilestoneFlash`) triggers at x3, x5, x8 — set in `triggerNearMiss()` after increment
- `ralph/` is gitignored — use `git add -f ralph/prd.json` to force-add PRD updates
