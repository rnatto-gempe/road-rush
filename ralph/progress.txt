## Codebase Patterns
- gameState is module-scoped (not on window) — browser evaluate can't access it; use screenshots or debug mode (D key) to verify game state
- ctx.createRadialGradient(cx, cy, r1, cx, cy, r2) is the pattern for vignette overlays — both fuel and speed vignettes use this
- All visual overlays (vignettes, flashes) use fillRect(0, 0, CANVAS_WIDTH, CANVAS_HEIGHT) after setting fillStyle to the gradient
- Render order in playingState.render(): road → traffic/items → player → particles → overlays → HUD (draw HUD last so overlays don't darken it)
- The game runs at 60fps fixed timestep (FIXED_DT = 1/60); scrollSpeed ramps from 200 to 800 px/s over the run
