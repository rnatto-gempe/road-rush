## Codebase Patterns
- FSM states: titleState, playingState, explosionState, gameOverState — all have onEnter/onExit/update/render
- titleState (not startState) is the initial state — "Press Enter to Start" transitions to playingState
- `keys` object is global: set keys['ArrowLeft']/keys['ArrowRight'] to control player movement
- `resizeCanvas()` computes displayWidth/displayHeight via letterbox math — canvas is centered using flexbox
- Touch button divs must use `position:fixed` (not absolute) since they overlay the full viewport
- `{ passive: false }` required on touch event listeners to allow e.preventDefault()
- Touch button positioning: left = (innerWidth - displayWidth) / 2, top = (innerHeight - displayHeight) / 2 + displayHeight * 0.80
